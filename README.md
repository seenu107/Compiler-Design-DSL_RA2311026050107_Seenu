# Mini Compiler Project

## Problem Title
Intermediate Code – Temporary Variable Growth Analysis

## Project Description
This project implements a mini compiler for a simple arithmetic Domain Specific Language (DSL) using Flex, Bison, C, and Python. The compiler performs lexical analysis, syntax analysis, and intermediate code generation. Python and Pandas are used to analyze temporary variable growth and generate optimization readiness flags.

## Tools Used
- Flex
- Bison
- GCC
- C Programming
- Python
- Pandas

## Features
- Lexical Analysis
- Syntax Analysis
- Intermediate Code Generation
- Temporary Variable Analysis
- Is_Temp_Heavy Flag Generation

## Sample Input
x = 5 + 3;

## Sample Output
t1 = 5 + 3
x = t1
